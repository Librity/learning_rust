# Learning Rust

## `rustc` commands

```bash
$ rustc src/main.rs
$ ./main
```

## `cargo` commands

To create a new project:

```bash
$ cargo new <project_name>
```

To create a new project in current directory:

```bash
$ cargo init
```

To build and run code (debug):

```bash
$ cargo run
```

To just build code (debug):

```bash
$ cargo build
```

To build production binaries:

```bash
$ cargo build --release
```
